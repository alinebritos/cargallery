
  # Car Gallery Website Design

  This is a code bundle for Car Gallery Website Design. The original project is available at https://www.figma.com/design/QpUPxniUVzA4p6oRczPkqx/Car-Gallery-Website-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Environment setup

  Before your first commit, configure Supabase credentials locally. See [docs/ENVIRONMENT.md](docs/ENVIRONMENT.md).

  ```bash
  cp .env.example .env   # then fill VITE_SUPABASE_ANON_KEY
  ```
